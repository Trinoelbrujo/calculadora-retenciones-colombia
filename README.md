# 🇨🇴 Calculadora de Retenciones Tributarias Colombia 2026

![Version](https://img.shields.io/badge/version-2026-blue)

Calculadora web para determinar retenciones tributarias en Colombia según normativa 2026.

## 📋 Características

- ✅ Cálculo de retención en la fuente
- ✅ Cálculo de IVA y retención de IVA
- ✅ Cálculo de ICA
- ✅ UVT 2026: $52,374
- ✅ Suspensión Decreto 572/2025 (Consejo de Estado)
- ✅ Base mínima de 27 UVT
- ✅ Adaptada para móviles
- ✅ Modo oscuro automático
- ✅ Funciona sin internet

## 🚀 Usar en línea

## 📱 Instalación en móvil

1. Abre el enlace en Chrome/Safari
2. Toca "Añadir a pantalla de inicio"
3. ¡Úsala como una app!
